## Data Science / Machine Learning Projects:

1. [Identify Different Types of Cell Nuclei in a Colon Cancer Sample using Deep Learning](https://github.com/shehzadahmedqureshi/Identify-Different-Types-Of-Cell-Nuclei-In-A-Colon-Cancer-Sample-Using-Deep-Learning/blob/main/Predict.ipynb): This project uses deep learning (PyTorch) to classify different types of cell nuclei in colon cancer histopathology images. Designed and trained a Deep Neural Network (CNN) model for a multi‑class classification problem on 64x64 images with cell nuclei at the center. Performed regularization, hyperparameter tuning, and data augmentation to improve results.
   Tools Used: Python, Pytorch, Scikit‑Learn, Matplotlib, Seaborn
   [Kaggle Competition Link](https://www.kaggle.com/shahzadqureshi/competitions)
2. [Explainable Travel Buddy Recommender Systems](https://github.com/shehzadahmedqureshi/Explainable-Travel-Buddy-Recommender-System/blob/main/Recommender_System_Models.ipynb): An experimental recommender system that suggests compatible travel buddies based on demographics, interests, and lifestyle preferences. Used TF-IDF, CountVectorizer, Word2Vec, and BERT algorithms to find matching people based on the data available.
3. [Subreddit Post Sentiment Classification](https://github.com/shehzadahmedqureshi/Subreddit-Post-Sentiment-Classification/blob/main/psc.ipynb): Sentiment analysis in contextual problems involves identifying the text and classifying it into social sentiments to get subjective information. Different brands use sentiment analysis to get an overview of wider public opinions on different topics from their social media pages. Moreover, sentiment analysis, also known as emotion Artificial Intelligence or opinion mining, is a very popular tool being used by many E-commerce brands for understanding a large number of reviews of their products, as it helps quickly get an overview of public opinion without going to read all the reviews. In this project, a dataset from Reddit is provided. Reddit provides deep discussion on different topics that are not very common in other social networks. Reddit has threads that demonstrate the topics of discussion. A thread may contain different posts from different users. The task is to correctly classify these posts for sentiment analysis. This project explores sentiment classification of Reddit posts using classical NLP and machine learning techniques.
4. [Explainable Hostel Recommender System](https://github.com/shehzadahmedqureshi/Explainable-Hostel-Recommender-System/blob/main/Explainable_Hostel_Recommender_System.ipynb): This project uses an Irish hostels dataset and combines recommendation techniques with simple, human-readable explanations so users can understand why particular hostels are suggested.
5. [Feature Selection for a dataset of predicting central neuropathic pain (PCNP)](https://github.com/shehzadahmedqureshi/Feature-Selection-For-Predicting-Central-Neuropathic-Pain/blob/main/fs.ipynb): In this project, a dataset of predicting central neuropathic pain (PCNP) is provided, which has 432 features. The task is to apply different feature selection methods to this dataset so that the overall performance can be increased. Three methods of feature selection are performed on this dataset so that the performance can be increased. These three feature selection methods are filtering, wrapper, and embedding methods. For this experiment, both techniques of embedding methods, i.e., L1, Lasso, and L2, Ridge, are used, and then the results are compared in the end.
6. [Content-Based-Hostel-Recommender-System](https://github.com/shehzadahmedqureshi/Content-Based-Hostel-Recommender-System/blob/main/Hostel_Recommender_System.ipynb): Created a content-based recommender system for hostels in Ireland that recommends similar hostels by matching the contextual features and multi-rating data of other hostels with the anchor hostel.
7. [Kaggle's Titanic Competition](https://github.com/shehzadahmedqureshi/Kaggle-s-Titanic-Competition/blob/main/titanic.ipynb): The goal of this project was to predict what sorts of people were likely to survive in the disaster.
8. [Name-Based-Hostel-Recommender-System](https://github.com/shehzadahmedqureshi/Name-Based-Hostel-Recommender-System/blob/main/hostel_recommender_system_based_on_hostel_name.ipynb): This recommender system uses the name of a hostel to recommend similar hostels.
9. [Support Vector Machines: Predict a Pulsar Star](https://github.com/shehzadahmedqureshi/Predict-A-Pulsar-Star/blob/main/support_vector_machines.ipynb): Our main goal is to predict pulsar candidates to facilitate rapid analysis using SVM.
10. [Predicting Black Friday Purchase Prices](https://github.com/shehzadahmedqureshi/Predicting-Black-Friday-Purchase-Prices/blob/main/black_friday.ipynb): The challenge is to predict purchase prices of different products purchased by customers based on historical purchase data.
11. [Market Bucket Optimization](https://github.com/shehzadahmedqureshi/Market-Bucket-Optimization/blob/main/apriori_groceries.ipynb): The challenge is to create association rules using the Apriori algorithm to optimize the bucket of a grocery store.
12. [Customer Segments using K-means](https://github.com/shehzadahmedqureshi/Customer-Segments-Using-K-Means/blob/main/customer_segments_kmeans.ipynb): This project applies the K-Means clustering algorithm to a customer dataset to discover distinct customer segments.
